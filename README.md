# 抖娃福袋工作台（用户使用说明）

## 这个软件能干啥

这是一个面向运营场景的桌面客户端，核心作用是：

1. 统一管理多个账号
2. 自动筛选可参与的直播福袋
3. 自动执行参与流程（可按策略过滤）
4. 实时查看账号状态、参与进度和收益数据
5. 保留操作日志和历史记录，便于复盘

简单说：它把“多账号盯直播 + 手动点参与”变成“集中管理 + 自动执行”。

## 怎么用（用户版）

### 第一步：进入软件并完成授权

1. 打开客户端
2. 进入“授权中心”
3. 按页面提示选择“试用”或“激活”
4. 授权成功后进入主界面

### 第二步：添加账号并扫码登录

1. 点击“添加账号”
2. 软件会弹出该账号的登录窗口
3. 用手机扫码完成登录
4. 登录成功后，账号会出现在“账号矩阵”中

### 第三步：启动任务

1. 在账号卡片上点击“启动”
2. 软件开始自动筛选并参与任务
3. 可随时点击“暂停”停止该账号

### 第四步：按需调策略

在“设置中心”可以调整常用策略，例如：

- 是否过滤粉丝团门槛
- 是否只参与实物类
- 是否优先大红包
- 停留时长与其他执行参数

修改后会立即生效。

### 第五步：看结果

你可以在主界面直接看到：

- 在线账号数
- 今日参与次数
- 今日收益
- 中签次数
- 实时运行日志与账号状态

## 用了之后你能得到什么

1. 节省人力：减少手动盯盘和重复点击
2. 提升效率：多账号并行，统一管理
3. 过程可见：每个账号的状态和日志都可追踪
4. 结果可量化：参与量、收益、中奖情况可直接查看

注意：软件提升的是执行效率，不承诺业务收益或中奖结果。
